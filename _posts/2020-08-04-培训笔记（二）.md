## 消防培训
### 四懂
#### 懂得岗位的火灾危险性
```
主因：明火
不规范使用电器及电器性能老化，例如插线板
```
#### 懂得预防火灾的措施
```
对于插线板，有6个问题可能引发火灾
1、过载
2、短路
3、灰尘
4、接触不良
5、插座保护壳损坏
6、线皮老化开裂
```

```
对于电脑
1、进排风口长期未清理
2、未关闭电源
```

```
对于饮水机
1、无干烧保护
2、短路
3、外壳损坏
4、线路老化
防范措施：
1、下班关电
2、定期检修
```

```
对于照明
1、线路老化
2、灯管下方近距离放置易燃杂物
防范措施：
1、灯管下方1米内，禁止放置货物
2、下班关灯
```

```
其他消防隐患：
防虫防鼠
```
