### 3、国际业务简介
```
1、政策性强
    （1）政策种类繁多，变动相对频繁
    （2）监管要求严格，强调三原则的落实
2、专业性强
    （1）国际结算惯例
    （2）贸易融资产品
    （3）政策解读研判
    （4）一定外语水平
3、操作复杂
    （1）资料审核复杂
    （2）业务流程繁琐
```

#### 主要分类
```
1、国际收支申报
2、汇兑（结汇、售汇、套汇）
3、外币汇款
4、我行账户行
5、托收
6、信用证
7、融资贸易
```

#### 贸易融资的特点
```
1、期限相对较短
2、与交易过程紧密结合
3、业务具有自偿性，风险相对可控
```
